🔐 Código do Steam Guard necessário

O bot está aguardando o código do Steam Guard. Verifique seu e-mail ou app do Steam e cole abaixo.

Configurar credenciais do bot
⚠️ Sem credenciais o bot opera em SIMULAÇÃO (não envia trades reais).
Para modo real, preencha apenas 3 campos obrigatórios:
1. Conta (login da Steam)
2. Senha
3. API key (pegue aqui)

O shared_secret e identity_secret são opcionais — sem eles, o bot pede o código do Steam Guard manualmente (campo acima) e salva um refresh token para logins futuros.

Credencial de pagamento (Mercado Pago)

Cole aqui o Access Token da sua conta Mercado Pago. Ele fica salvo no servidor — assim você troca de máquina sem mexer no código.

ℹ️ Não é a sua chave PIX. O Access Token é uma credencial de API (começa com APP_USR- em produção ou TEST- em teste). Pegue em: Mercado Pago → Suas integrações → Credenciais.
